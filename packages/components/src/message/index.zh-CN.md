## API

图标的属性说明如下：

| 属性  | 说明                                                                              | 类型             | 默认值 | 版本 |
| ----- | --------------------------------------------------------------------------------- | ---------------- | ------ | ---- |
| size  | 图标大小，当传入的是 number 类型时，会加上`px`单位后缀；也可以直接使用 style 设置 | string \| number | `1em`  |      |
| color | 图标颜色，可以直接使用 style 设置                                                 | string           | false  |      |

支持原生 HTML 的其他所有属性。
