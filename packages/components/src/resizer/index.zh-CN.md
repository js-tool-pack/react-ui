## API

Resizer 的属性说明如下：

| 属性      | 说明       | 类型                                   | 默认值   | 版本 |
| --------- | ---------- | -------------------------------------- | -------- | ---- |
| placement | 拖动条位置 | 'left' \| 'right' \| 'top' \| 'bottom' | 'bottom' | --   |

拖动条支持原生 HTMLElement 的其他所有属性。
